## 진단 퀴즈 오답 (1회차)

### ❌ Q2 — CNCF 성숙도: Graduated 아닌 것 찾기
- 카테고리: CNCF 생태계·거버넌스
- 내 답: c (Envoy)
- 소크라테스 힌트:
  - Envoy는 언제 Graduated가 되었을까요? 서비스 메시 데이터플레인의 사실상 표준으로 자리 잡은 프로젝트가 아직 Sandbox/Incubating에 머물러 있을까요?
  - 보기 4개 중 **비교적 최근에 등장했고, 이벤트 기반 오토스케일링**을 담당하는 프로젝트는 무엇인가요? 그 프로젝트의 현재 성숙도 단계는?
- 상태: 🔴 미정정
- 복습 예정: D+1

### ❌ Q6 — 오토스케일링 짝 중 "잘못된" 것
- 카테고리: 아키텍처·원칙
- 내 답: b (VPA — 리소스 요청/제한 조정)
- 소크라테스 힌트:
  - b의 "VPA가 Pod의 리소스 요청/제한을 조정한다"는 설명은 실제로 맞는 설명일까요, 틀린 설명일까요?
  - d의 "KEDA — 노드의 CPU 클럭을 조정"을 다시 읽어 보세요. KEDA가 실제로 조정하는 대상은 노드의 클럭 속도인가요, 아니면 **이벤트 기반으로 워크로드(Pod) 수**인가요?
  - 문제는 "잘못된 짝"을 고르는 것입니다. 네 보기 중 설명 자체가 사실과 다른 것은?
- 상태: 🔴 미정정
- 복습 예정: D+1

### ❌ Q7 — 관측성 3대 축

- 내 답: c (CPU, Memory, Disk)
- 소크라테스 힌트:
