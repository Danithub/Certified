# → KCNA 2주 합격 로드맵 (CKA/CKAD 보유자 맞춤)

> 목표: 2주 안에 KCNA 합격 → Kubestronaut 3부작(KCNA→KCSA→CKS)의 첫 관문 통과
원칙: 이미 아는 K8s 코어는 빠르게, 낯선 "생태계/원칙"은 확실하게, 모의고사로 마무리
> 

---

## ✓ 사용법

- 각 Day의 체크박스를 완료할 때마다 체크하세요.
- 하루 목표 학습량: **1-1.5시간** (진도 빠르면 앞당겨 응시 OK)
- 강의는 **1.5-2배속**, 아는 부분은 건너뛰기.

| 항목 | 내용 |

| --- | --- | --- |

| Container Orchestration | 28% | Networking, Security, Troubleshooting, Storage |
| Cloud Native Application Delivery | 16% | Application Delivery, Debugging |
| Cloud Native Architecture | 12% | Observability, 생태계·원칙, 커뮤니티·협업 |

> · CKA/CKAD 보유자에게 실제 득점 포인트는 44%/28%(이미 앎)가 아니라 **16%+12%(생태계·GitOps·관측성·원칙)**입니다.

## 🧠 공부 방법 (핵심 5)

1. 강의는 배속, 아는 K8s 코어는 스킵 → 낯선 부분에서만 정지
2. 12%+16% 도메인(생태계/GitOps/관측성)에 시간 더 투자
3. 인풋(강의)보다 아웃풋(문제풀이) 비중 높이기
4. 헷갈리는 프로젝트 이름·역할은 아래 치트시트로 암기
5. 실전 전 모의고사 **85% 이상** 안정화가 목표
